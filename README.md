## Getting Started

### Prerequisites

I build this project with the following setup:

- macOS Monterey v12.0 Beta
- node v16.7.0
- yarn v1.22.11
- npm v7.20.3

### Development

1. create your `.env` file based on the `.env.example`

2. install all necessary dependencies

```bash
yarn install
````

3. start app

```bash
yarn start
```

### Tests

run prettier check

```bash
yarn prettier-check
```

run all tests

```bash
yarn test
```
